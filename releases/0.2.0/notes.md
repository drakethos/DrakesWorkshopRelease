## Install

### First time
1. Download **DrakesWorkshop-0.2.0-Windows.zip**
2. Unzip to a folder
3. Run `DrakesWorkshop.exe` and enter your DWK- key

### Already installed
1. Download **DrakesWorkshop-0.2.0-Update.zip**
2. Unzip into the same folder (overwrite)
3. Leave `DrakesWorkshop.exe` and the rest of `data_ModuleGameWorkshop_*` alone
